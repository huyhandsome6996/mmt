Thử thách Lập trình

Thực hành Code Tự luận

#1 Duyệt số chẵn

Viết chương trình sử dụng for hoặc while để in ra các số chẵn từ 1 đến n (với n có thể tự chọn).

#2 Tính tổng

Viết chương trình tính tổng dãy số: S = 1 + 2 + 3 + ... + n và in ra kết quả cuối cùng.

#3 Bảng cửu chương

Viết đoạn code in ra Bảng cửu chương của một số nguyên bất kỳ (Ví dụ: in bảng nhân 5).

#4 Vòng lặp vô tận

Dùng while True để tạo vòng lặp bắt nhập mật khẩu. Nếu biến mk = "1234" thì break.

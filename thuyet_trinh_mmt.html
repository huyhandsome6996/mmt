Phần 3.1: Mở rộng mạng - Căn cứ Quy hoạch IP

Giải thích chi tiết bước tính IP cho Link R0-R1 và Net 3 dành cho người mới bắt đầu

Bước 8 Tính Link R0-R1 (Cáp Serial)

Nhu cầu IP: Chỉ cần chuẩn xác 2 IP
Số bit Host (h): 2h - 2 ≥ 2 → h = 2
Prefix & Mask: 32 - 2 = /30
Địa chỉ Mạng: Nối tiếp .158 → 175.25.37.160/30
Router 0 .161
Router 1 .162

Bước 9 Tính Net 3 (50 PC + 1 Server)

Tổng IP cần: 50 + 1 (Srv) + 1 (GW) = 52 IP
Số bit Host (h): 2h - 2 ≥ 52 → h = 6
Prefix & Mask: 32 - 6 = /26
Bước nhảy: 26 = 64 → .0, .64, .128, .192
Địa chỉ Mạng: Mốc trống → 175.25.37.192/26
Gateway .193
Server .194
DHCP (50 PC) .195 → .244

Bảng Quy Hoạch IP (IP Planning) Toàn Hệ Thống

Vùng mạng Chức năng Network / Mask Gateway Dải IP khả dụng
Net 1 Switch 0 (PC0, 1) 175.25.37.96/27 .97 .98 → .126
Net 2 Switch 1 (PC2, 3) 175.25.37.128/27 .129 .130 → .158
Link R0-R1 Cáp Serial WAN 175.25.37.160/30 Không .161(R0), .162(R1)
Net 3 (Mới) Switch 2 (50 PC, 1 Server) 175.25.37.192/26 .193 Server: .194
DHCP: .195 → .244

Phần 3.2: Cấu hình Interface & Định tuyến tĩnh

Giải thích siêu chi tiết ý nghĩa từng dòng lệnh cấu hình (CLI) cho Router

CLI: Router0
Router> enable
! Bật chế độ Đặc quyền (#) để có quyền quản trị
Router# configure terminal
! Vào chế độ Cấu hình toàn cục (Global Config)
Router(config)# no ip domain-lookup
! Tắt dò tìm DNS, giúp Router không bị treo khi gõ sai lệnh
Router(config)# interface g0/0/0
! Đi vào cài đặt cổng LAN 1 (Cổng nối xuống Net 1)
Router(config-if)# ip address 175.25.37.97 255.255.255.224
! Gán IP (.97) làm Gateway cho máy tính ở Net 1 đi qua
Router(config-if)# no shutdown
! Bật nguồn mở cổng lên (Mặc định bị tắt)
Router(config)# interface g0/0/1
! Đi vào cài đặt cổng LAN 2 (Cổng nối xuống Net 2)
Router(config-if)# ip address 175.25.37.129 255.255.255.224
! Gán IP (.129) làm Gateway cho Net 2
Router(config-if)# no shutdown
! Bật nguồn mở cổng lên xanh
Router(config)# interface s0/1/0
! Đi vào cổng cáp Serial (WAN) nối sang Router 1
Router(config-if)# ip address 175.25.37.161 255.255.255.252
! Gán IP liên kết P2P (Dùng Subnet Mask của /30)
Router(config-if)# no shutdown
! Bật nguồn cổng cáp Serial
Router(config-if)# exit
! Thoát khỏi chế độ cài đặt cổng (Interface)
Router(config)# ip route 175.25.37.192 255.255.255.192 175.25.37.162
! Cú pháp: [Mạng muốn tới] [Mask mạng tới] [Cổng Next-hop]. Lệnh này dạy Router 0 biết cách gửi thư sang Net 3 (50 PC) thông qua trạm kế tiếp là .162
CLI: Router1
Router> enable
! Bật chế độ Đặc quyền (#)
Router# configure terminal
! Vào chế độ Cấu hình toàn cục
Router(config)# no ip domain-lookup
! Ngăn tìm kiếm DNS rác tránh lỗi treo lệnh
Router(config)# interface g0/0/0
! Chọn cổng LAN 3 (Cổng cắm Switch 50 PC)
Router(config-if)# ip address 175.25.37.193 255.255.255.192
! Gán IP Gateway cho mạng Net 3
Router(config-if)# no shutdown
! Bật nguồn mở cổng lên
Router(config)# interface s0/1/0
! Chọn cổng Serial WAN (Cáp nối về Router 0)
Router(config-if)# ip address 175.25.37.162 255.255.255.252
! Gán IP cho đầu dây cáp bên này (/30)
Router(config-if)# no shutdown
! Bật nguồn cổng
Router(config-if)# exit
! Thoát ra ngoài cấu hình toàn cục
Router(config)# ip route 175.25.37.96 255.255.255.224 175.25.37.161
! Dạy Router 1 biết đường gửi thư ngược về Net 1 thông qua cửa tiếp theo là .161
Router(config)# ip route 175.25.37.128 255.255.255.224 175.25.37.161
! Dạy Router 1 biết đường gửi thư ngược về Net 2 thông qua cửa tiếp theo là .161
